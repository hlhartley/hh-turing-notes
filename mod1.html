Mod 1 Notes

CSS: Flexbox

Manipulates spacing and position of contents (vertically and horizontally - 1 dimensional)
- note: good for simpler layouts and isolated components


Tips:

- Can put a flexbox inside a grid layout
- Don't "flex" your body
- Parent container should have "display: flex" property
- Center content by using "justify-content: center" and "align-items: center" properties on parent container
- Child element should have height and width

https://css-tricks.com/snippets/css/a-guide-to-flexbox/

CSS: Grid

Manipulates spacing and position of contents (2D)
- note: good for larger page layouts


Tips:

- Can put a flexbox inside a grid layout
- Can "flex" your body
- Columns must be specified
- Rows do not need to be specified
- Numbers can be positive or negative
- Can use special unit (fr)
- "grid-auto-flow: dense" property fills all empty spaces
- Grid template areas must work with grid-area (must define "area names")

Link - A Complete Guide to Grid:

https://css-tricks.com/snippets/css/complete-guide-grid/
