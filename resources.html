HTML/CSS: Import Customized Font

Changes font to a customized font style of your choosing
- note: linking the font to your HTML page takes less loading time than linking to your CSS page using @import rule


1. Find a font on Google Fonts website
2. Optional: select and customize font (font-weight styles)
3. Copy and paste HTML code in the <head> section of your HTML document
4. Add "font-family" property in CSS file and change 'Merriweather' to the name of your new font


Link - Google Fonts:

https://fonts.google.com/


CSS Code:

* { font-family: 'Merriweather', serif; }

CSS: normalize.css

Defines specific default CSS values that will be displayed standardized across all browsers


1. Copy and paste the normalize.css code from the link below in a new, separate file from your CSS (in your root directory)
2. Link the normalize.css file in your HTML in the <head> section of your HTML document before the link to your regular css file


Link - normalize.css File Code:

https://necolas.github.io/normalize.css/8.0.1/normalize.css


HTML Code:

<link rel="stylesheet" href="normalize.css">

CSS: Defining General CSS Styles (*) / Box-sizing: border-box

* defines default CSS values that will apply to the entire page.
Box-sizing: border-box property allows us to incude the padding and border in an element's total width and height


1. Paste CSS code at top of CSS file


CSS Code:

* { box-sizing: border-box; }

HTML: Meta Tag

Makes your site viewer friendly for mobile devices


1. Insert HTML in the <head> section of your HTML document


HTML Code:

<meta name=“viewport” content=“width=device-width, initial-scale=1”>
